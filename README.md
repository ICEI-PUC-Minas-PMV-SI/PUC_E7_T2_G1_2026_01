# Modelagem Preditiva para Identificação de Risco de Inadimplência em Clientes de Crédito

`CURSO: Sistemas de Informação`

`DISCIPLINA: Projeto - Pesquisa e Experimentação em Sistemas de Informação`

`SEMESTRE: 7º`

O objetivo do projeto é analisar dados realcionados ao perfil financeiro de clientes para identificar padrões de comportamento financeiro que estejam associados ao risco de inadimplência. O foco é desenvolver um modelo capaz de prever a propabilidade de um devedor não cumprir com os prazos de suas obrigações financeiras.

Com base probabilidade, o projeto poderá apoiar processos de tomada de decisão em instituições como bancos, fintechs e cooperativas financeiras.
